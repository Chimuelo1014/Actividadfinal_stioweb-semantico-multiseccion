Header

Descripción

La etiqueta <header> define un encabezado para un documento o sección. Suele contener logotipos, menús o títulos.

Ejemplo:


    <header>
        <h1>Bienvenidos</h1>
    </header>

Nav

Descripción

La etiqueta <nav> se usa para contener enlaces de navegación dentro de la página.

Ejemplo:


    <nav>
        <ul>
            <li><a href="index.html">Inicio</a></li>
        </ul>
    </nav>

Section

Descripción

La etiqueta <section> representa una sección temática dentro de un documento, como capítulos, pestañas o bloques separados.

Ejemplo:


    <section>
        <h2>Servicios</h2>
        <p>Ofrecemos desarrollo web y diseño UX.</p>
    </section>

Article

Descripción

La etiqueta <article> se usa para contenido independiente y auto-contenido como entradas de blog, noticias o publicaciones.

Ejemplo:


    <article>
        <h2>Nuevo lanzamiento</h2>
        <p>Hoy lanzamos nuestro nuevo producto...</p>
    </article>

Footer

Descripción

La etiqueta <footer> define el pie de página de una sección o documento. Puede incluir información de contacto o derechos de autor.

Ejemplo:


    <footer>
        <p>© 2025 Mi Sitio Web</p>
    </footer>
